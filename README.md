# ✨ About The Project

The Hungry Oven is a premium frontend restaurant experience focused on visual storytelling, immersive interactions, smooth animations, and modern UI architecture.

Instead of building a traditional food website, the goal was to create a cinematic digital experience that feels dynamic, interactive, and visually memorable.

This project combines:

* Modern frontend engineering
* Motion-driven user experience
* Responsive architecture
* Interactive design systems
* Premium visual aesthetics
